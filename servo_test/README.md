# Servo Test

Basic servo control via Serial. Send an angle (0–180) to move the servo.

## Hardware
- Arduino
- Servo on pin 9

## Pin Assignments
| Component | Pin |
|-----------|-----|
| Servo | 9 |

## Usage
1. Open Serial Monitor at 9600 baud
2. Type an angle between 0 and 180 and send
3. Servo moves to that angle and prints confirmation
